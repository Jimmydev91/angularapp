Angular challange

Detta är ett projekt gjort på några dagar för att skapa ett backend och frontend. Backend med C# och .NET. Frontend med Angular. Här används JSON token (JWT) för att sköta och kolla inloggning är korrekt mot backend. Bootstrap används för responstivitet och skapa en snygg och enhetlig design. Uppgiften gick ut på att skapa en sida för att skapa böcker. Där man kan skapa, redigera och ta bort böcker. Sen även ha en sida med sina favoritcitat.
